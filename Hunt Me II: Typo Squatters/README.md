# Hunt Me II – Typosquatters | Incident Investigation

This repository documents a full incident investigation conducted as part of
the TryHackMe room “Hunt Me II – Typosquatters”.

The investigation tracks a real-world attack scenario from typosquatted
software download through credential compromise, lateral movement, and
ransomware impact.

---

## Key Skills Demonstrated

- Threat hunting and timeline reconstruction
- Detection of typosquatting attacks
- PowerShell abuse analysis
- Credential harvesting and lateral movement tracking
- Persistence and privilege escalation analysis
- SOC-style incident reporting

---

## Repository Structure

- report.txt – Chronological incident report
- screenshots/ – Evidence screenshots for each investigation step
- README.md – Project overview

All analysis was performed in a controlled lab environment for educational and portfolio purposes.
