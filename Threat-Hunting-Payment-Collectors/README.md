# Hunt Me I – Payment Collectors | Incident Investigation

This repository contains a full incident investigation based on the
TryHackMe room “Hunt Me I – Payment Collectors”.

The investigation follows a phishing-led compromise from initial access
through enumeration and data exfiltration preparation.

## Key Focus Areas

- Phishing attachment analysis
- PowerShell abuse
- Living-off-the-Land techniques
- Active Directory enumeration
- Data staging and exfiltration preparation
- SOC-style reporting

## Repository Structure

- report.txt – Full chronological incident report
- screenshots/ – Evidence screenshots for each investigation step
- README.md – Project overview

All work was performed in a controlled lab environment.
