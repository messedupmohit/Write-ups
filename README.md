# Cybersecurity Write-ups

This repository contains cybersecurity write-ups focused on blue team and SOC-related investigations.  
The write-ups document practical security analysis scenarios such as phishing email investigations, log analysis, and detection-based assessments.

The purpose of this repository is to demonstrate hands-on investigation skills, analytical thinking, and the ability to document security findings in a clear and structured manner.

---

## Skills Demonstrated

- Phishing email investigation
- Email header analysis
- Social engineering detection
- Log analysis and threat identification
- Indicator of Compromise (IOC) identification
- Incident investigation and reporting
- SOC-style documentation
